markdown语法

本文规定：整个文档题目采用一级标题，每一大章题目采用二级标题，以此类推

一.标题

说明：
前面有n个#就是n级标题
编码：
### 例如这是三级标题
效果：

例如这是三级标题

二.字体

说明：
一组**包括的内容为加粗字体，*为倾斜字体，***为加粗倾斜字体，~~为删除线字体
编码：
**这是加粗字体**，*这是倾斜字体*，***这是加粗倾斜字体***，~~这是删除线字体~~
效果：
这是加粗字体，这是加粗字体，这是加粗倾斜字体，这是删除线字体

三.引用

说明：
一个>后面的内容为引用内容
编码：
>这是引用的内容
效果：

这是引用的内容

四.分割线

说明：
3个以上的-或*，为分割线，注意：与前面内容空一行，否则会变成标题
编码：
---，***
